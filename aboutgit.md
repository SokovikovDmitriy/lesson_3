# Создание сохранений

## Добавитть файл к коммиту

* **git add file_name**

## Добавление всех файлов к коммиту

* **git add .**
## Зафиксировать изменения

* **git commit -m "some info"**
## Добавить все файлы к коммиту

* **git commit -a -m "some info"**
